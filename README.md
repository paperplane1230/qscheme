#qscheme
A simple tiny scheme interpreter referring to http://norvig.com/lispy2.html.

#Usage
    python3 scheme.py

#Example
    python3 scheme.py <examples/test.scm
or

    python3 scheme.py
then input `(load "examples/test.scm")` in the interactive environment

    > (load "examples/test.scm")
